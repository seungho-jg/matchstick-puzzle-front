# Matchstick Puzzle
React로 만든 성냥개비 퍼즐 사이트입니다. <a href="https://matchstick-puzzle.com/">바로가기</a>

# 개발 기간
24.11 ~ 24.01 (약 1달)

# 주요 기능
로그인, 퍼즐 제작, 퍼즐 풀기, 레벨 시스템, 스킨 시스템, 리더보드 등

# 기술 스택
React18, Konva.js, zustand, tanstack-query, tailwindCSS, Zod
